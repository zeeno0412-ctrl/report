MISO RAG · 측정 방법

RAGAS

RAG 시스템의 답변 품질을 자동 평가하는 오픈소스 프레임워크

청크 기반 검색 품질 측정

  • Precision@K — 검색된 Top-K 청크 중 실제 관련 청크 비율
  • Hit Rate — 정답 청크가 Top-K 안에 있는지 여부
  • MRR — 정답 청크가 몇 번째 순위에 위치하는지

Precision은 RAGAS 자동 측정이 납득이 안 가서
→ 직접 골든 답 심어서 평가했음

AI 기반 답변 품질 측정

  • Faithfulness — 답변이 검색된 청크에 충실한지
    낮으면 모델이 멋대로 생성 (할루시네이션)
  • Answer Relevancy — 답변이 질문과 얼마나 관련 있는지

THEN SUBSTR(origin_bizpl_cd, 2) ELSE origin_bizpl_cd END AS clean_storecd,

        
COUNT(DISTINCT CONCAT(pos_no, '-', CAST(sale_seq AS STRING))) AS valid_cust_cnt

        
FROM gsr_offline.gsucdw_tb_ft_cvs_tr_item

        
WHERE oper_dt = '2026-01-15' AND sale_sp = '1' AND item_purch_qty > 0

        
AND sku_cancel_yn = 'N' AND deal_sp IN ('01','33','61','49') AND goods_cd <> ''

        
GROUP BY 1

        
) C ON TRIM(M.storecd) = TRIM(C.clean_storecd)

        
WHERE M.part_ln = '김재영' -- OFC명

        
ORDER BY net_sales DESC
